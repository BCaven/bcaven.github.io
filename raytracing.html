Ray tracing Development

Built a Monte Carlo ray tracer in C++ based on the tutorials RaytracingInAWeekend and RaytracingTheNextWeek.

In addition to the base functionality from the tutorials, I implemented multi-core chunk based rendering, triangle-based mesh structures with custom UV maps, and a custom file format to import meshes into the raytracer from Blender.

Further work is being done on writing the ray tracer in HLSL.
